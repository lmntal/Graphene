# UNYO-UNYO

## 操作方法
* ノードをドラッグするとノードを移動でき、画面をドラッグすると画面を移動できます
* Shiftを押しながらノードをクリックするとこで複数選択ができます
* ノードを選択している状態で右クリックすると、深さによるノードの色付けができます
